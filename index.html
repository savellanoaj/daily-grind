This JavaScript project creates a single page that rotates unique colors and content for each weekday (Sunday to Saturday) into the page. In order to rotate content, we'll use data extracted from the Date object, and then use a conditional (for example a switch block) to determine which HTML is appropriate to show at a given date. We also have the ability to choose the day through the suggested links.
